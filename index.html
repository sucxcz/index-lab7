Практическая работа CSS
  1. Порядок и приоритет стилей для одного тега: встроенные стили > внутренние стили > внешние стили.
  2. Для изменения одного элемента на одной странице лучше использовать встроенные стили.
  3. При разработке больших проектов лучше использовать внешние стили.
  4. Недостатки каждого способа:
      5. Встроенные: сложно обслуживать, возможно дублирование стилей.
      Внутренние: могут замедлить загрузку страницы.
      Внешние: могут вызвать проблемы с кэшированием и производительностью.
